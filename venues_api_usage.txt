"*" - Обязательные параметры

1) Получить список заведений(удовлетворяющих определенным критериям и 
	отсортированным согласно заданным параметрам).
	URL: http://bronimesto.by:8080/BookingServer2/venues_list
	Тип запроса: GET
	Параметры:
		lat, lng - GPS-координаты текущего положения пользователя(широта и долгота соответственно)
		limit - ограничение размеров списка заведений
		has_free_tables(boolean) - показывать только заведения, где есть свободные места
		has_wifi(boolean) - показывать только заведения, где есть WiFi
		take_credit_cards(boolean) - показывать только заведения, где принимают кредитные карты
		has_outdoors_places(boolean) - показывать только заведения, где есть места на улице
		type(String) - показывать заведения определенного типа(Кафе, Рестораны, Бары, Ночные клубы,...)
		cuisine(String) - показывать заведения с определенной кухней(Белорусская, Итальянская, Французская,...)
		rating_order - Сортировать в порядке убывания рейтинга заведения
			
	Возвращает: Список заведений, удовлетворяющих критериям, в формате json.
	
	Примечание: 
		Если не указан порядок сортировки по рейтингу - список сортируется по расстоянию до местоположения 
			пользователя, в порядке увеличения расстояния(Ближайшие заведения - в начале списка).
			
2) Установить количество свободных столиков в заведении: 
	URL: http://bronimesto.by:8080/BookingServer2/set_free_tables	 
	Тип запроса: POST
	Параметры:
		adminUser * - Логин администратора заведения
		adminPassword * - Пароль администратора заведения
		venueId * - Id заведения
		freeTables * - Количество свободных столиков
	Возвращает - json
		status(success/failure)
		error(Если статус = failure) - Сообщение об ошибке
